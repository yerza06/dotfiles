# Палитра Flexoki для интерфейса

[Flexoki](https://stephango.com/flexoki) — это теплая, дружелюбная палитра, разработанная Стефаном Анго. Она вдохновлена аналоговыми цветами и бумагой, обеспечивая отличную читаемость и комфорт для глаз.

## Базовые цвета (Base Colors)

Используются для создания структуры интерфейса: фонов, текста, границ и теней.

| Группа | Цвет | Hex | Применение |
| :--- | :--- | :--- | :--- |
| **Dark Theme BG** | `black` | `#100F0F` | Основной фон темной темы |
| | `base-950` | `#1C1B1A` | Поверхности (карточки, меню) в темной теме |
| | `base-900` | `#282726` | Границы и выделение в темной теме |
| **Intermediate** | `base-500` | `#878580` | Вторичный текст, иконки |
| | `base-300` | `#B7B5AC` | Разделители, неактивные элементы |
| **Light Theme BG** | `base-50` | `#F2F0E5` | Поверхности (карточки, меню) в светлой теме |
| | `paper` | `#FFFCF0` | Основной фон светлой темы («бумага») |

## Акцентные цвета (Accent Colors)

Каждый цвет имеет 13 градаций (от 50 до 950).

### Основные уровни использования:
- **400**: Рекомендуется для светлой темы (яркие акценты, кнопки).
- **600**: Рекомендуется для темной темы (текстовые акценты, важные элементы).
- **50-200**: Фоновые выделения, мягкие подложки.
- **700-950**: Глубокие тени, контрастный текст на светлом фоне.

## Примеры комбинаций

1. **Текст (Ink):**
   - На светлом фоне (`paper`): используйте `black` или `base-900`.
   - На темном фоне (`black`): используйте `base-100` или `base-50`.

2. **Состояния элементов:**
   - **Hover:** Смещение на 1 шаг по шкале (например, с 400 на 500).
   - **Active/Pressed:** Смещение на 2 шага (например, с 400 на 600).
   - **Disabled:** Используйте `base-300` или `base-600` с пониженной прозрачностью.

3. **Информационные блоки:**
   - Фон блока: уровень `100` соответствующего цвета.
   - Текст/иконка внутри: уровень `600` или `700`.
